Vowels & Diphthongs

  • Basic Vowels (Session 10 optimization):
  • A = a
  • E = e
  • U = i (OPTIMIZED - most frequent!)
  • O = o
  • OE = u
  • Diphthongs:
  • AEU = aj (HIGH FREQ - plural adj)
  • AOEU = ej
  • OEU = oj (HIGH FREQ - plural noun)
  • AOU = uj
  • AU = aŭ
  • EU = eŭ

Special Markers

  • # = proper nouns (REQUIRED)
  • KPH- = grammatical endings (inflection)
  • WHR- = vowel-initial syllables + derivational suffixes (phonetic)
  • WR/KWR/TWR/WR*/TPWHR = correlatives (grammatical)
  • -Z = affix marker (derivational)
  • KPHO = {^o} noun ending
  • KPHA = {^a} adj ending
  • KPHAS = {^as} present
  • U or I = {^i} infinitive (short form)
  • O* = {^on} accusative (Session 10!)
  • O*EU = {^ojn} plural+acc (Session 10!)
  • All names must start with # (e.g., #PHAR/KU = Marko)
  • KWREL = kiel (how/in what manner)
    WROPB = ion (something-ACC)
    TWRAPL = tiam (then)

Right-Hand Clusters (2-key)

  • -FT = st
  • -PL = m
  • -PB = n
  • -BG = k
  • -FP = v
  • -GS = h
  • -TS = c
  • -BD = br
  • -TD = tr (inverted - T-D outputs t-r)
  • -PD = dr (reinterpreted - P-D outputs d-r)
  • -PG = gl (reinterpreted - P-G outputs g-l)
  • -FD = pr (reinterpreted - F-D outputs p-r) - Session 14!
  • Very common: PROFD = propr (own/proper). Can't use -PR (violates steno order)

Right-Hand Clusters (3-key)

  • -FPL = sm
  • -RPB = rn
  • -PBL = mb
  • -PLG = lm
  • -PLT = mp
  • -PBG = ng
  • -BGS = ks
  • -PBS = ns
  • -FBG = sk (F=s Lapwing rule)
  • -BLG = lk (direct mapping: alk, bulk, falk, kalk, melk, silk, etc.)
  • -PLD = pl (reinterpreted)
  • -FRT = str
  • -FRP = rv
  • -FRD = rf
  • -FBL = lv (letter reinterpretation - pelv, salv, etc.)
  • -FLT = lf (letter reinterpretation - golf, didelf)
  • -RBS = br (October)
  • Note: -FRB freed in Session 10 (now uses FAEUR pattern with AEU vowel cluster)

Right-Hand Clusters (4-key)

  • -RPBT = ntr (inverted)
  • -RPBL = mbr
  • -FPLT = mpl (inverted)
  • -FPBT = nst
  • -PBGT = nkt
  • -FBGT = kst (F=s Lapwing rule)
  • -PBGS = gn (regn)
  • -BLGS = kl (reinterpreted: bukl, kunikl, pekl, tikl)
  • Note: Letter inversion - stroke follows steno order,
    but outputs letters in different order
    Examples: montri, centro, vintro, membro, ekzemplo, tempo, punkt

Right-Hand Clusters (5-key)

  • -FRBGT = kstr (inverted - dekstr, ekstr, ekstrem)
  • Complex 5-key cluster using letter inversion
    Physical stroke: F-R-B-G-T outputs: k-s-t-r

Grammatical Endings

  • Inflectional endings (finalize words): KPH- versions or short alternatives
  • Short alternatives are ultra-efficient! Use them for speed.
  • Word Class Endings:
  • KPHO = {^o} (noun) | short: O
  • KPHA = {^a} (adjective) | short: A
  • KPHE = {^e} (adverb) | short: E
  • KPHU = {^i} (infinitive) | short: U
  • Verb Tenses:
  • KPHAS = {^as} (present) | short: AS
  • KPHUS = {^is} (past) | short: US
  • KPHOS = {^os} (future) | short: OS
  • KPHOES = {^us} (conditional) | short: OES
  • KPHOE = {^u} (imperative) | short: OE
  • Plural:
  • KPHAEU = {^aj} (plural adjective) | short: AEU
  • KPHOEU = {^oj} (plural noun) | short: OEU
  • Accusative (Session 10 optimization!):
  • KPHOPB = {^on} (singular acc) | short: O* (2 keys!)
  • KPHA*PB = {^an} (adj acc) | short: A* (2 keys!)
  • KPHO*EUPB = {^ojn} (plural+acc) | short: O*EU (4 keys!)
  • KPHA*EUPB = {^ajn} (plural adj+acc) | short: A*EU (4 keys!)
  • Active Participles:
  • KPHAPBT = {^ant^} (present) | shorts: APBT, A*T
  • KPHUPBT = {^int^} (past) | shorts: UPBT, *UT
  • KPHOPBT = {^ont^} (future) | shorts: OPBT, O*T
  • KPHOEPBT = {^unt^} (conditional - unofficial) | shorts: OEPBT, O*ET
  • Passive Participles:
  • KPHAT = {^at^} (present) | short: AT
  • KPHUT = {^it^} (past) | short: UT
  • KPHOT = {^ot^} (future) | short: OT
  • KPHOET = {^ut^} (conditional - unofficial) | short: OET
  • Other:
  • TPH* or *PB = {^n} (accusative marker alone)
  • Key optimization: O*/A* accusative shortcuts save 50%+ keystrokes! (Session 10)

Derivational Suffixes

  • Compositional forms (use with roots): attach to root, wait for grammatical ending
  • Most suffixes have brief right-hand alternatives shown in parentheses
  • Size & Quality:
  • EG = {eg^} (augmentative) | brief: -GS
  • ET = {et^} (diminutive) | brief: -T
  • A*TS = {aĉ^} (pejorative) | brief: *-TS
  • A*FBZ = {^aĵ^} (concrete thing) | brief: *-F
  • *-FB or A*EUF = {^uĵ^} (excellent quality)
  • Person & Gender:
  • OEL = {ul^} (person characterized by) | brief: -LZ
  • UPB = {in^} (feminine) | brief: -B
  • UP or *-P = {ip^} (nonbinary)
  • *UTS or TH* = {^iĉ^} (masculine)
  • UFT = {ist^} (profession) | brief: -FT
  • UFPL = {ism^} (doctrine) | brief: -S
  • APB = {an^} (member) | brief: AB
  • Affectionate:
  • TPH-FBZ = {^nj^} (feminine affectionate) | brief: -FP
  • P-FB = {^pj^} (nonbinary affectionate)
  • TH*FBZ or TPR-TS = {^ĉj^} (masculine affectionate)
  • Place & Collection:
  • AOEU or WH = {ej^} (place)
  • AR = {ar^} (collection) | brief: R-
  • UPBG = {ing^} (holder) | brief: -PG
  • Things & Tools:
  • UL = {il^} (tool) | brief: -L
  • UD = {id^} (offspring) | brief: -DZ
  • AOE = {^uj} (container/country) | brief: -FB
  • Abstract & Quality:
  • ETS = {ec^} (abstract quality) | brief: -TS
  • ER = {er^} (single unit) | brief: -R
  • EFRT = {estr^} (leader) | brief: -FR
  • Possibility & Tendency:
  • EBL = {ebl^} (possible) | brief: EB
  • EPL = {em^} (tendency) | brief: -PL
  • UPBD = {ind^} (worthy) | brief: -PD
  • EPBD = {end^} (must be done) | brief: -PBD
  • Action & Change:
  • AD = {ad^} (continuous action) | brief: TK
  • UG = {ig^} (make/cause) | brief: -G
  • *UG = {iĝ^} (become) | brief: *-G
  • Numbers:
  • OBL = {obl^} (multiple) | brief: OB
  • OPB = {on^} (fraction) | brief: -F
  • OP = {op^} (collective) | brief: -P
  • Other:
  • OEPL = {um^} (indefinite relation) | brief: -SZ

Prefixes

  • Attach to following root: {prefix^} + root + ending
  • PH or PHAL = {mal^} (opposite - very common!)
  • RE = {re^} (again/back)
  • TKP or TKPE = {ge^} (both sexes together)
  • PW or PWO = {bo^} (in-law relationship)
  • T- or TKUS = {dis^} (separation/apart)
  • EBG = {ek^} (sudden action/start)
  • SK or EBGS = {eks^} (former/ex-)
  • PRA = {pra^} (ancient/primitive)
  • TP or TPU = {fi^} (disgraceful/morally bad)
  • PHUS = {mis^} (wrongly/incorrectly)
  • Note: Also available: TPH/TPHE = ne (negation), PHEPL = mem (self)

Correlatives (Tabelvortoj) - Session 11 Update!

  • Base patterns (WR system):
  • WR- = i- (indefinite: io, iu, ie, iel...)
  • KWR- = ki- (interrogative: kio, kiu, kie, kiel...)
  • TWR- = ti- (demonstrative: tio, tiu, tie, tiel...)
  • WR*- = ĉi- (universal - BRIEF: drops T, saves 2 keys!)
  • TPWHR- = neni- (negative: nenio, neniu, nenie, neniel...)
  • Extended WR* brief for A/O vowels:
  • WRA* = ĉia, WRA*L = ĉial, WRA*PB = ĉian, WRA*PL = ĉiam
  • WRO* = ĉio, WRO*PB = ĉion, WRO*PL = ĉiom
  • Endings combine with bases:
  • -O = thing (io, kio, tio, ĉio, nenio)
  • -U = person (iu, kiu, tiu, ĉiu, neniu)
  • -E = place (ie, kie, tie, ĉie, nenie)
  • -EL = manner (iel, kiel, tiel, ĉiel, neniel)
  • -AL = reason (ial, kial, tial, ĉial, nenial)
  • -APL = time (iam, kiam, tiam, ĉiam, neniam)
  • -OPL = quantity (iom, kiom, tiom, ĉiom, neniom)
  • -A = quality (ia, kia, tia, ĉia, nenia)
  • -ES = possessive (ies, kies, ties, ĉies, nenies)
  • Example: KWREL = kiel (how), WRA*PL = ĉiam (always - brief!), WRO* = ĉio (everything)
  • WR = correlatives (grammatical) | WHR = syllable linkers (phonetic)

Brief Patterns - Session 14

  • Possessive Adjectives: -PLS Pattern
  • Pattern: PRONOUN_CONSONANT + -PLS (single stroke for all possessives!)
  • S-PLS = sia (4 keys)
  • S*-PLS = ŝia (5 keys)
  • PH-PLS = mia (5 keys)
  • W-PLS = via (4 keys)
  • HR-PLS = lia (5 keys)
  • TPH-PLS = nia (6 keys)
  • TKP*-PLS = ĝia (7 keys)
  • R-PLS = ria (4 keys)
  • UPLS = ilia (4 keys)
  • OPLS = onia (4 keys)
  • Special:
  • TPHOEPLS = unua (first - ordinal, single stroke!)
  • -PLS = "possessive" - consistent, memorable pattern for high-frequency words

Steno Order

  • Full: #STKPWHRAO*EUFRPBLGTSDZ
  • Left: #STKPWHR*
  • Vowels: AO EU
  • Right: FRPBLGTSDZ
  • Note: Letters must follow this order within a stroke
