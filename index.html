Circumflexed Letters (*)

  • S* = ŝ (left)
  • TH* = ĉ (left)
  • TKP* = ĝ (left)
  • WH* = ĵ (left)
  • H* = ĥ (left)
  • -*G = ĝ (right)
  • -*GS = ĥ (right)
  • -*TS = ĉ (right)
  • -*S = ŝ (right)
  • -*FB = ĵ (right)

Vowels & Diphthongs

  • Basic Vowels (Session 10 optimization):
  • A = a
  • E = e
  • U = i (OPTIMIZED - most frequent!)
  • O = o
  • OE = u
  • Diphthongs:
  • AEU = aj (HIGH FREQ - plural adj)
  • AOEU = ej
  • OEU = oj (HIGH FREQ - plural noun)
  • AOU = uj
  • AU = aŭ
  • EU = eŭ

Special Markers

  • # = proper nouns (REQUIRED)
  • KPH- = grammatical endings (inflection)
  • WHR- = vowel-initial syllables + derivational suffixes (phonetic)
  • WR/KWR/TWR/WR*/TPWHR = correlatives (grammatical)
  • -Z = affix marker (derivational)
  • KPHO = {^o} noun ending
  • KPHA = {^a} adj ending
  • KPHAS = {^as} present
  • U or I = {^i} infinitive (short form)
  • O* = {^on} accusative (Session 10!)
  • O*EU = {^ojn} plural+acc (Session 10!)
  • All names must start with # (e.g., #PHAR/KU = Marko)
  • KWREL = kiel (how/in what manner)
    WROPB = ion (something-ACC)
    TWRAPL = tiam (then)

Right-Hand Clusters (2-key)

  • -FT = st
  • -PL = m
  • -PB = n
  • -BG = k
  • -FP = v
  • -GS = h
  • -TS = c
  • -BD = br
  • -TD = tr (inverted - T-D outputs t-r)
  • -PD = dr (reinterpreted - P-D outputs d-r)
  • -PG = gl (reinterpreted - P-G outputs g-l)

Right-Hand Clusters (3-key)

  • -FPL = sm
  • -RPB = rn
  • -PBL = mb
  • -PLG = lm
  • -PLT = mp
  • -PBG = ng
  • -BGS = ks
  • -PBS = ns
  • -FBG = sk (F=s Lapwing rule)
  • -BLG = lk (direct mapping: alk, bulk, falk, kalk, melk, silk, etc.)
  • -PLD = pl (reinterpreted)
  • -FRT = str
  • -FRP = rv
  • -FRD = rf
  • -FBL = lv (letter reinterpretation - pelv, salv, etc.)
  • -FLT = lf (letter reinterpretation - golf, didelf)
  • -RBS = br (October)
  • Note: -FRB freed in Session 10 (now uses FAEUR pattern with AEU vowel cluster)

Right-Hand Clusters (4-key)

  • -RPBT = ntr (inverted)
  • -RPBL = mbr
  • -FPLT = mpl (inverted)
  • -FPBT = nst
  • -PBGT = nkt
  • -FBGT = kst (F=s Lapwing rule)
  • -PBGS = gn (regn)
  • -BLGS = kl (reinterpreted: bukl, kunikl, pekl, tikl)
  • Note: Letter inversion - stroke follows steno order,
    but outputs letters in different order
    Examples: montri, centro, vintro, membro, ekzemplo, tempo, punkt

Right-Hand Clusters (5-key)

  • -FRBGT = kstr (inverted - dekstr, ekstr, ekstrem)
  • Complex 5-key cluster using letter inversion
    Physical stroke: F-R-B-G-T outputs: k-s-t-r

Grammatical Endings (KPH- prefix)

  • KPHO = {^o} (noun)
  • KPHA = {^a} (adjective)
  • KPHE = {^e} (adverb)
  • KPHU = {^i} (infinitive) | Short: U or I
  • KPHAS = {^as} (present) | Short: AS
  • KPHUES = {^is} (past) | Short: IS
  • KPHOS = {^os} (future) | Short: OS
  • KPHOES = {^us} (conditional) | Short: US
  • KPHOE = {^u} (imperative) | Short: OE
  • KPHAEU = {^aj} (plural adj)
  • KPHOEU = {^oj} (plural noun)
  • KPHOPB = {^on} (accusative) | Short: O*
  • KPHA*PB = {^an} (adj accusative) | Short: A*
  • KPHA*EUPB = {^ajn} (plural adj acc) | Short: A*EU
  • KPHO*EUPB = {^ojn} (plural noun acc) | Short: O*EU
  • KPHAPBT = {^ant^} (active present participle)
  • KPHUEPBT = {^int^} (active past participle)
  • KPHOPBT = {^ont^} (active future participle)
  • KPHOEPBT = {^unt^} (active conditional - unofficial)
  • KPHAT = {^at^} (passive present participle)
  • KPHUET = {^it^} (passive past participle)
  • KPHOT = {^ot^} (passive future participle)
  • KPHOET = {^ut^} (passive conditional - unofficial)
  • Short alternatives save keystrokes! Session 10 adds O*/A*/O*EU/A*EU for accusative.

Correlatives (Tabelvortoj) - Session 11 Update!

  • Base patterns (WR system):
  • WR- = i- (indefinite: io, iu, ie, iel...)
  • KWR- = ki- (interrogative: kio, kiu, kie, kiel...)
  • TWR- = ti- (demonstrative: tio, tiu, tie, tiel...)
  • WR*- = ĉi- (universal - BRIEF: drops T, saves 2 keys!)
  • TPWHR- = neni- (negative: nenio, neniu, nenie, neniel...)
  • Extended WR* brief for A/O vowels:
  • WRA* = ĉia, WRA*L = ĉial, WRA*PB = ĉian, WRA*PL = ĉiam
  • WRO* = ĉio, WRO*PB = ĉion, WRO*PL = ĉiom
  • Endings combine with bases:
  • -O = thing (io, kio, tio, ĉio, nenio)
  • -U = person (iu, kiu, tiu, ĉiu, neniu)
  • -E = place (ie, kie, tie, ĉie, nenie)
  • -EL = manner (iel, kiel, tiel, ĉiel, neniel)
  • -AL = reason (ial, kial, tial, ĉial, nenial)
  • -APL = time (iam, kiam, tiam, ĉiam, neniam)
  • -OPL = quantity (iom, kiom, tiom, ĉiom, neniom)
  • -A = quality (ia, kia, tia, ĉia, nenia)
  • -ES = possessive (ies, kies, ties, ĉies, nenies)
  • Example: KWREL = kiel (how), WRA*PL = ĉiam (always - brief!), WRO* = ĉio (everything)
  • WR = correlatives (grammatical) | WHR = syllable linkers (phonetic)

Steno Order

  • Full: #STKPWHRAO*EUFRPBLGTSDZ
  • Left: #STKPWHR*
  • Vowels: AO EU
  • Right: FRPBLGTSDZ
  • Note: Letters must follow this order within a stroke
