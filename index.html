Special Keys

Key Combinations (Left)

  • TP = f
  • TK = d
  • PW = b
  • PH = m
  • TPH = n
  • HR = l
  • TKP = g
  • TH = c
  • SH = z
  • WH = j

Circumflexed Letters (*)

  • S* = ŝ (left)
  • TH* = ĉ (left)
  • TKP* = ĝ (left)
  • WH* = ĵ (left)
  • H* = ĥ (left)
  • -*G = ĝ (right)
  • -*GS = ĥ (right)
  • -*TS = ĉ (right)
  • -*S = ŝ (right)
  • -*FB = ĵ (right)

Diphthongs

  • AOE = i
  • EFB = ej
  • AFB = aj
  • OFB = oj
  • UFB = uj
  • AU = aŭ
  • EU = eŭ
  • OU = ŭ

Special Markers

  • # = proper nouns (REQUIRED)
  • SKWR- = grammatical endings (inflection)
  • KWR- = vowel-initial syllables
  • (corr. start key) + WHR- + ending = correlatives
  • -Z = affix marker (derivational)
  • SKWRO = {^o} noun ending
  • SKWRA = {^a} adj ending
  • SKWRAS = {^as} present
  • SKWRAOE = {^i} infinitive
  • All names must start with # (e.g., #PHAR/KAOE = Marko)
  • KWHREL = kiel (how/in what manner)
    WHROPB = ion
    TWHRAPL = tiam

Right-Hand Clusters (2-key)

  • -FT = st
  • -PL = m
  • -PB = n
  • -BG = k
  • -FP = v
  • -GS = h
  • -TS = c
  • -FPL = sm
  • -RPB = rn
  • -PLG = lm
  • -BD = br

Right-Hand Clusters (3-key)

  • -PBG = ng
  • -PBTS = nc
  • -RPBT = ntr (inverted)
  • Note: Letter inversion - stroke follows steno order,
    but outputs letters in different order

Right-Hand Clusters (4-key)

  • -RPBL = mbr
  • -FPLT = mpl
  • -FPBT = nst
  • Examples: membro, ekzemplo, konstrui

Steno Order

  • Full: #STKPWHRAO*EUFRPBLGTSDZ
  • Left: #STKPWHR*
  • Vowels: AO EU
  • Right: FRPBLGTSDZ
  • Note: Letters must follow this order within a stroke
