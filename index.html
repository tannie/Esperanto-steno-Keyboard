Special Keys

Key Combinations (Left)

  • TP = f
  • TK = d
  • PW = b
  • PH = m
  • TPH = n
  • HR = l
  • TKP = g
  • TH = c
  • SH = z
  • WH = j

Circumflexed Letters (*)

  • S* = ŝ (left)
  • TH* = ĉ (left)
  • TKP* = ĝ (left)
  • WH* = ĵ (left)
  • H* = ĥ (left)
  • -*G = ĝ (right)
  • -*GS = ĥ (right)
  • -*TS = ĉ (right)
  • -*S = ŝ (right)
  • -*FB = ĵ (right)

Diphthongs

  • AOE = i
  • EFB = ej
  • AFB = aj
  • OFB = oj
  • UFB = uj
  • AU = aŭ
  • EU = eŭ
  • OU = ŭ

Special Markers

  • # = proper nouns (REQUIRED)
  • KPH- = grammatical endings (inflection)
  • KWR- = vowel-initial syllables + optional derivational suffixes
  • (corr. start key) + WHR- + ending = correlatives
  • -Z = affix marker (derivational)
  • KPHO = {^o} noun ending
  • KPHA = {^a} adj ending
  • KPHAS = {^as} present
  • KPHAOE = {^i} infinitive
  • All names must start with # (e.g., #PHAR/KAOE = Marko)
  • KWHREL = kiel (how/in what manner)
    WHROPB = ion
    TWHRAPL = tiam

Right-Hand Clusters (2-key)

  • -FT = st
  • -PL = m
  • -PB = n
  • -BG = k
  • -FP = v
  • -GS = h
  • -TS = c
  • -BD = br

Right-Hand Clusters (3-key)

  • -FPL = sm
  • -RPB = rn
  • -PLG = lm
  • -PLT = mp
  • -PBG = ng
  • -PBTS = nc
  • -PBLG = nk

Right-Hand Clusters (4-key)

  • -RPBT = ntr (inverted)
  • -RPBL = mbr
  • -FPLT = mpl (inverted)
  • -FPBT = nst
  • Note: Letter inversion - stroke follows steno order,
    but outputs letters in different order
    Examples: membro, ekzemplo, konstrui, tempo, punkt

Grammatical Endings (KPH- prefix)

  • KPHO = {^o} (noun)
  • KPHA = {^a} (adjective)
  • KPHE = {^e} (adverb)
  • KPHAOE = {^i} (infinitive) | Short: I
  • KPHAS = {^as} (present) | Short: AS
  • KPHAOES = {^is} (past) | Short: IS
  • KPHOS = {^os} (future) | Short: OS
  • KPHUS = {^us} (conditional) | Short: US
  • KPHU = {^u} (imperative) | Short: U
  • KPHAFB = {^aj} (plural adj)
  • KPHOFB = {^oj} (plural noun)
  • KPHOPB = {^on} (accusative)
  • KPHAFPB = {^ajn} (plural adj acc)
  • KPHOFPB = {^ojn} (plural noun acc)
  • KPHAPBT = {^ant^} (active present participle)
  • KPHAOEPBT = {^int^} (active past participle)
  • KPHOPBT = {^ont^} (active future participle)
  • KPHUPBT = {^unt^} (active conditional - unofficial)
  • KPHAT = {^at^} (passive present participle)
  • KPHAOET = {^it^} (passive past participle)
  • KPHOT = {^ot^} (passive future participle)
  • KPHUT = {^ut^} (passive conditional - unofficial)
  • Short alternatives (I, AS, IS, OS, US, U) save keystrokes for common verbs

Correlatives (Tabelvortoj)

  • Base patterns:
  • WHR- = i- (indefinite: io, iu, ie, iel...)
  • KWHR- = ki- (interrogative: kio, kiu, kie, kiel...)
  • TWHR- = ti- (demonstrative: tio, tiu, tie, tiel...)
  • TWHR*- = ĉi- (universal: ĉio, ĉiu, ĉie, ĉiel...)
  • TPWHR- = neni- (negative: nenio, neniu, nenie, neniel...)
  • Endings combine with bases:
  • -O = thing (io, kio, tio, ĉio, nenio)
  • -U = person (iu, kiu, tiu, ĉiu, neniu)
  • -E = place (ie, kie, tie, ĉie, nenie)
  • -EL = manner (iel, kiel, tiel, ĉiel, neniel)
  • -AL = reason (ial, kial, tial, ĉial, nenial)
  • -APL = time (iam, kiam, tiam, ĉiam, neniam)
  • -OPL = quantity (iom, kiom, tiom, ĉiom, neniom)
  • -A = quality (ia, kia, tia, ĉia, nenia)
  • -ES = possessive (ies, kies, ties, ĉies, nenies)
  • Example: KWHREL = kiel (how), TWHR*APL = ĉiam (always)

Steno Order

  • Full: #STKPWHRAO*EUFRPBLGTSDZ
  • Left: #STKPWHR*
  • Vowels: AO EU
  • Right: FRPBLGTSDZ
  • Note: Letters must follow this order within a stroke
